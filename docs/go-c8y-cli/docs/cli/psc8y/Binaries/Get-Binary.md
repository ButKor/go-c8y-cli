## SYNOPSIS
Download binary

## SYNTAX

```
Get-Binary
	[-Id] <Object[]>
	[-Raw]
	[-OutputFile <String>]
	[-OutputFileRaw <String>]
	[-Proxy]
	[-NoProxy]
	[-Timeout <Double>]
	[-Session <String>]
	[-Output <String>]
	[-View <String>]
	[-AsHashTable]
	[-AsPSObject]
	[-Flatten]
	[-Compress]
	[-Pretty]
	[-NoColor]
	[-Color]
	[-Confirm]
	[-ConfirmText <String>]
	[-WithError]
	[-SilentStatusCodes <String>]
	[-Dry]
	[-DryFormat <String>]
	[-Workers <Int32>]
	[-Delay <Int32>]
	[-MaxJobs <Int32>]
	[-Progress]
	[-NoLog]
	[-LogMessage <String>]
	[-Select <String[]>]
	[<CommonParameters>]
```

## DESCRIPTION
Download a binary stored in Cumulocity and display it on the console.
For non text based binaries or if the output should be saved to file, the output parameter should be used to write the file directly to a local file.

## EXAMPLES

### EXAMPLE 1
```
Get-Binary -Id $Binary.id
```

Get a binary and display the contents on the console

### EXAMPLE 2
```
Get-Binary -Id $Binary.id -OutputFile ./download-binary1.txt
```

Get a binary and save it to a file
