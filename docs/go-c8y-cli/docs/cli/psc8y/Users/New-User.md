## SYNOPSIS
Create user

## SYNTAX

```
New-User
	[[-UserName] <Object[]>]
	[[-FirstName] <String>]
	[[-LastName] <String>]
	[[-Phone] <String>]
	[[-Email] <String>]
	[-Enabled]
	[[-Password] <String>]
	[-SendPasswordResetEmail]
	[[-CustomProperties] <Object>]
	[[-Tenant] <Object>]
	[-Data <Object>]
	[-NoAccept]
	[-ProcessingMode <String>]
	[-Force]
	[-Template <String>]
	[-TemplateVars <String>]
	[-Raw]
	[-OutputFile <String>]
	[-OutputFileRaw <String>]
	[-Proxy]
	[-NoProxy]
	[-Timeout <Double>]
	[-Session <String>]
	[-Output <String>]
	[-View <String>]
	[-AsHashTable]
	[-AsPSObject]
	[-Flatten]
	[-Compress]
	[-Pretty]
	[-NoColor]
	[-Color]
	[-Confirm]
	[-ConfirmText <String>]
	[-WithError]
	[-SilentStatusCodes <String>]
	[-Dry]
	[-DryFormat <String>]
	[-Workers <Int32>]
	[-Delay <Int32>]
	[-MaxJobs <Int32>]
	[-Progress]
	[-NoLog]
	[-LogMessage <String>]
	[-Select <String[]>]
	[<CommonParameters>]
```

## DESCRIPTION
Create a new user so that they can access the tenant

## EXAMPLES

### EXAMPLE 1
```
New-user -Username "$Username" -Password "$NewPassword"
```

Create a user
